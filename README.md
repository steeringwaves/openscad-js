# openscad-js

![workflow](https://github.com/steeringwaves/openscad-js/actions/workflows/test.yml/badge.svg)

Javascript/typescript based code generator for openscad.

## dev

```sh
npm install

# bundle with esbuild
npm run build

# run
npm run bin

# type help to see a list of available commands
```
